# Zalith Launcher Modern UI Replica Plan

## Project Overview
Create a modernized Android Minecraft launcher based on ZalithLauncher with enhanced animations, smoother theming, and Material Design 3 aesthetics while excluding promotional assets.

## Technology Stack
- **Language**: Kotlin 2.0.21
- **Min SDK**: 26 (Android 8.0)
- **Target SDK**: 34 (Android 14)
- **Build System**: Gradle 8.x with Kotlin DSL
- **UI**: ViewBinding + Material Design 3
- **Architecture**: MVVM with Clean Architecture principles
- **Animations**: Android Property Animations + MotionLayout

## Key Enhancements Over Original

### 1. Modern Material Design 3 Theming
- Dynamic color support with Material You
- Elevated surfaces with proper shadow handling
- Smooth light/dark theme transitions with animation
- Refined color palette (keeping teal accent, modernizing other colors)

### 2. Enhanced Animations
- Shared element transitions between screens
- Animated vector drawables for icons
- Spring-based physics animations for natural motion
- Ripple effects with custom extendable material
- Animated FAB and bottom navigation
- Staggered list animations for version lists

### 3. Smoother Theme Transitions
- Smooth theme switching (instant color changes → animated transitions)
- Window background color transitions
- Proper state preservation during transitions
- Animated status bar and navigation bar colors

### 4. Excluded Promotional Assets
- Remove sponsor-related UI and features
- Remove donation/sponsor dialogs
- Clean up sponsor-related fragments
